## üí° About the Project
**Dream Garage Hunter (DGH)** is advanced, lightweight analytical software tailored to the needs of professional car dealers, importers, and car dealerships. 

DGH is not an ordinary search engine. It is an **analytical sniper** that treats the used car market like a stock exchange. It automatically extracts data, utilizes advanced mathematics to calculate the *Fair Price*, and instantly highlights market anomalies (undervalued listings), allowing investors to profit immediately from arbitrage.

## üöÄ Core Features (Pillars)

### 1. Cold Calculation (Data-Driven Deal Score)
A proprietary statistical algorithm that removes emotions from the equation. It uses an **exponential depreciation curve** as a function of mileage, calibrating live against collected listings (discarding 5% of extreme anomalies). Listings receive a profitability ranking reminiscent of video games (Legendary, Epic, Rare).

### 2. Bulletproof Scraper & Smart DOM Parsing
A data extraction system resilient to portal code changes:
* Advanced regular expressions bypassing rotating titles and textual noise (e.g., "1/6", "Promoted").
* Seamless, automatic currency conversion utilizing live NBP (National Bank of Poland) exchange rates via API.
* Prepared for Proxy server integration to protect against IP bans.

### 3. Professional, Minimalist Dashboard
* Real-time chart rendering visualizing free trend space.
* Generation of a standardized Vehicle Financial Card.
* Instant export of calculated and evaluated results to a CSV (Excel) file.
